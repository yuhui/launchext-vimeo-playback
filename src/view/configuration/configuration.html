How this Extension works

  1. Create a new Rule or open an existing Rule, for example, one with a "Window Loaded" Event.
  2. (optional) Add the "Load Vimeo Player SDK script" Action.
  3. Add the "Enable video playback tracking" Action.
    If the "Load Vimeo Player SDK script" Action was not used, then this Action allows you to load the Vimeo Player SDK script.

Then add the appropriate Events in your Rules to track Vimeo video playback interactions.
Refer to each Event's table of Data Elements that you can use in your Rules' Conditions and Actions.

Use the appropriate Data Elements in your Rules to retrieve data about the Vimeo video playback interactions, like the player's state, current time, video URL, etc.
More information is available when setting up the Data Element.
