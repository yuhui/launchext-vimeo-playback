How this Action works

When this Action runs, it loads the Vimeo Player SDK script into the web page, if it has not been loaded yet.

This Action needs to be run only once on a web page, for example, with the Core extension's "Library Loaded" Event. Another use-case is to run it after the user has granted consent to be tracked.

If you have selected the "Load Vimeo Player SDK script if it hasn't been loaded yet" checkbox in the "Enable Vimeo Video Playback Tracking" Action, then you don't need to include this Action. (You can still include it, but nothing will happen.)
